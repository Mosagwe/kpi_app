OPENAI SETUP

Enable AI refinement

The app needs an OpenAI API key on the local server. A ChatGPT subscription does not automatically provide an API key.

  1. Create an API key at platform.openai.com/api-keys.
  2. Create a file named .env in the application folder.
  3. Add the following values:
OPENAI_API_KEY=your_actual_api_key
OPENAI_MODEL=gpt-5.5
OPENAI_PROXY_URL=http://proxy.company.com:8080
PORT=3010

Leave OPENAI_PROXY_URL blank when no proxy is required. Restart the app after saving. Secrets stay on the server and are never sent to browser storage.
